<h1>Projeto: Site Institucional do CEEP</h1>

<h2>Objetivo do Trabalho</h2>
<p>
Desenvolver um site institucional para o Centro Estadual de Educação Profissional (CEEP),
com o objetivo de divulgar informações sobre a escola, seus cursos, trajetória e canais
de contato, facilitando o acesso da comunidade às informações.
</p>

<h2> Como Executar o Site</h2>
<ol>
<li>Baixe ou clone o repositório</li>
<li>Abra a pasta no VS Code</li>
<li>Abra o arquivo <strong>index.html</strong></li>
<li>Execute com o navegador ou Live Server</li>
</ol>
